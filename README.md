## 📌 مميزات المشروع

- 🔍 البحث عن رقم التتبع (Tracking Code)
- 🧑‍💼 لوحة تحكم للإدارة (Admin Panel)
- ➕ إضافة شحنات جديدة
- 💾 حفظ البيانات محلياً (LocalStorage)
- 🎨 تصميم حديث ومتجاوب مع جميع الأجهزة

---

## 🛠️ التقنيات المستعملة

- HTML5
- CSS3 (Animations + Responsive Design)
- JavaScript (Vanilla JS)
- LocalStorage API

---

## ⚙️ طريقة التشغيل

1. قم بتحميل المشروع أو عمل Clone من GitHub
2. افتح ملف `index.html` في المتصفح
3. أو قم بنشره عبر GitHub Pages

---

## 🔐 لوحة التحكم

لفتح لوحة الإدارة:
- اضغط على زر 🔐 الموجود أسفل الصفحة
- قم بإدخال بيانات الشحنة الجديدة
- سيتم توليد كود تتبع تلقائياً

---

## 📦 ملاحظة مهمة

- البيانات تُحفظ فقط في المتصفح (LocalStorage)
- عند حذف بيانات المتصفح سيتم فقدان الشحنات
- لا يوجد سيرفر أو قاعدة بيانات حقيقية
